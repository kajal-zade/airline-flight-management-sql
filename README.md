# Airline Flight Management System (SQL)

## Project Overview
This project focuses on designing and analyzing an airline flight management database using SQL.

## Database Tables
- Flights
- Passengers
- Bookings
- Aircraft
- Airports

## Key Analysis Performed
- Flight occupancy analysis
- Revenue analysis
- Booking pattern analysis
- Identification of popular destinations
- Identification of low-occupancy flights

## SQL Concepts Used
- SELECT
- JOIN
- GROUP BY
- HAVING
- COUNT, SUM, AVG

## Tools Used
- MySQL
